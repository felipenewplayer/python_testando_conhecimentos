# Diferenças simples entre Regressão e Classificação

#| Tipo              | Saída (y)          | Exemplo                                     | Modelo usado                            |
| ----------------- | ------------------ | ------------------------------------------- | --------------------------------------- |
| **Regressão**     | Número contínuo    | Altura, preço, salário                      | `LinearRegression`                      |
| **Classificação** | Categoria / rótulo | Aprovado/Reprovado, Doente/Saudável, 0 ou 1 | `LogisticRegression`, `KNN`, `SVC`, etc |
